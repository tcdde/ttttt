- Animated statistics
- Online admission enquiry through WhatsApp
- Student/staff/community profile cards
- News & Events section
- Gallery category filters and lightbox
- Google Maps embed
- Donation page section
- English/Swahili language switch

- SEO meta tags, Open Graph tags and structured data
- Parent portal design
- Admin dashboard design

IMPORTANT SECURITY NOTE
The Parent Portal and Admin Dashboard are visual prototypes only.
They are not safe for real private information until connected to:
- Firebase Authentication or Supabase Auth
- Firestore / Supabase database
- Secure access rules
- Real hosting with HTTPS

PREVIEW
Extract the ZIP and open index.html.

PUBLISH
Upload all files and folders to Firebase Hosting, GitHub Pages, Netlify, Hostinger or another web host.

BEFORE PUBLISHING
- Replace sample profiles with approved names and biographies.
- Confirm all fees, donation methods and contact information.
- Replace the canonical URL in index.html after buying a domain.
- Obtain consent before publishing identifiable student stories or names.
